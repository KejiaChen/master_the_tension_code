# Mater the Tension

This is the repository is for reseach article "Master the Tension: Force-Driven Assembly of Deformable Linear Objects with Environmental Contacts".

This repository contains the following contents
- Transition Planning, consisting of [planning workflow](git@github.com:KejiaChen/master_the_tension_transition_plan.git) and custom [MoveIt 2 Task Concstructor]()
- Adaptive Fixing, consisting of [control framework for Franka Panda robot](git@github.com:KejiaChen/master_the_tension_mios.git)

The specific workflow and system requirement is specified in each module.
